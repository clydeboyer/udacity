HTML Basics

HTML Tags

HTML markup is made up of tags; an opening tag and closing tag. The primary tags we reviewed were as follows:

  • Bold tag
  • Italics tag
  • The link tag
  • The image tag
  • and the humble paragraph tag
Inline vs Block elements

Block elements have an invisible line around the element and can height and width.

Span tags are inline and Div elements are block level.

Document Structure

!DOCTYPE HTML opens the document and designates the document to follow as HTML. The head tag defines metadata for the document like title.
